# 2026-05-04

## Changed
- 为 `ConfirmationCard` 增加“自己输入”入口，提交时会取消当前确认并作为新的用户消息继续对话。
- 为 `update_settings` 确认卡接入自定义输入配置，允许用户在模型建议值之外继续补充自己的设置要求。
- 为 `insert_at_cursor` 确认卡接入自定义输入配置，允许用户在模型建议插入内容之外继续补充自己的文本。
- 为 `replace_selection` 确认卡接入自定义输入配置，允许用户在模型建议替换内容之外继续补充自己的文本。
- 为富文本编辑器的 Markdown 块节点补充真实源码行号映射，避免“插入对话”沿用渲染态文本行号。
- 将 `SelectionToolbar` 的文件引用行号切换为基于源码映射的真实 `startLine/endLine`，移除按 `textBetween()` 猜行号的旧逻辑。
- 将块级源码行号聚合细化为块内精确选区映射，修复多行段落中只选第 6 行却回退到第 5 行的问题。
- 为 `SelectionToolbar` 增加基于原始 Markdown 顶层 token 的源码行号回退映射，修复计划文档中因空行和未挂载源码 attrs 的块节点导致第 7 行回退为第 5 行的问题。
- 为聊天文件引用插入载荷补充 `renderStartLine` / `renderEndLine`，用于保留渲染视图中的用户感知行号。
- 扩展聊天文件引用 token 协议为 `{{#path start-end|renderStart-renderEnd}}`，并保持旧的 `{{#path start-end}}` 兼容可读。
